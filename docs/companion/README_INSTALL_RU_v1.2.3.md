Добавьте в Project Knowledge файлы из `00_AGENTIC_PIPELINE_INDEX_v1.2.3.md`.

Главный рабочий принцип:

```text
одна цель владельца
→ один короткий work brief
→ Antigravity сам определяет точные файлы
→ автоматический repair внутри scope
→ один итог
```

Companion не просит утверждать план, не просит сравнивать SHA и не выпускает новый task pack для того же work item.
