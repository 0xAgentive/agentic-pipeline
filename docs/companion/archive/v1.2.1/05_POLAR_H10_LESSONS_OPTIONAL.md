# Optional Example: Polar / H10 Lessons

Use only for H10 Athlete Cardio Lab or similar local health-adjacent products.

## Separation of concerns

Keep separate:

1. deterministic local analysis;
2. LLM Pack interpretation handoff;
3. LLM diary or narrative context.

LLM responses must never mutate deterministic metrics.

## Safety and validation

Use non-diagnostic wording:

- candidate event;
- quality-limited;
- insufficient data;
- implementation aligned;
- empirically unvalidated;
- discuss with a professional if repeated or symptomatic.

Do not equate:

- centralized threshold with validated threshold;
- documented formula with correct formula;
- implementation completion with empirical validation;
- ACC motion context with a stable workload/power proxy.

Use independent fields for implementation, verification, audit, empirical validation and ship status.

## Evidence level

Privacy, methodology, report and release work normally needs E3. Small UI wording changes may use E1/E2. Do not force E4 archival packaging on every development step.
