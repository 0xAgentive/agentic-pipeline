# v1.1.1b-r4 Productivity Guards

This is a stabilization patch. It does not introduce v1.2 architecture.

## Repair loop limits

- `/nextphase`: maximum one targeted repair pass after a failing check.
- `/fixcritical`: maximum two targeted repair passes.
- `/fastpatch`: no repair loop. If the final gate fails, stop and require `/auditphase` or `/nextphase`.

After the limit, mark the state as blocked and ask for human review or `/auditphase`.

## Fastpatch truth

`/fastpatch` is allowed only after the final diff exists and `scripts/Test-FastPatchAllowed.ps1 -RequireChanges` exits with code 0.

A clean preflight is not completion evidence.

## Hook truthfulness

Hook scripts shipped with this package are manual guard scripts unless `.agents/hooks.json` is explicitly enabled and verified in the local Antigravity workspace.

Do not claim active lifecycle hooks when `.agents/hooks.json` is empty.

## Evidence truth

Model-written evidence is only a pointer. Command exit codes, diffs, logs, test outputs, screenshots, and CI results are verification.
