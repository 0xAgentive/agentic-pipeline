# Requirement Drift Audit

Compare the latest user goal with:
- product contract;
- specification;
- implementation plan;
- phase state;
- tests and required artifacts.

Record unresolved deltas in `.agy/REQUIREMENTS_DELTA.md`.

Do not allow shipcheck to pass while a material requirement delta remains unresolved.
