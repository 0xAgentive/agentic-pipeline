# /visualqa

Do not implement features unless explicitly in a repair phase.

Required evidence:

- screenshot or walkthrough artifact;
- console error summary;
- tested route/scope;
- dark theme readability;
- dropdown readability;
- no raw i18n keys;
- no browser alert/confirm for app workflows;
- charts readable and not clipped.

Block `/shipcheck` if UI evidence is missing for UI projects.
