# Как пользоваться этим шаблоном

Эта папка — базовый шаблон проекта для Antigravity.

## Новый проект

1. Скопируй шаблон в новую папку.
2. Открой папку в Antigravity.
3. Запусти `/specdoc`.
4. Потом `/planonly`.
5. Потом `/nextphase` по одной фазе.

## Действующий проект

Если проект уже начат, сначала смотри:

```powershell
Get-Content .agy\PHASE_STATUS.json -Raw
git status --short
```

Если непонятно, что делать дальше:

```text
/auditphase
```

## Главное правило

Агент не должен сам перескакивать к следующей фазе. Он делает одну фазу, запускает проверки, обновляет state и останавливается.
