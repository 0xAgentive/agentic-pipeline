# Как пользоваться этим шаблоном

Эта папка содержит базовые файлы шаблона проекта Antigravity для Agentic Development Pipeline.

## Жизненный цикл рабочей области

Для любой рабочей области проекта, использующей этот пайплайн, запускайте воркфлоу в следующем порядке:

1.  **Спецификация требований**:
    ```text
    /specdoc
    ```
2.  **Планирование реализации**:
    ```text
    /planonly
    ```
3.  **Локальный аудит окружения**:
    ```text
    /auditphase
    ```
4.  **Пошаговая разработка**:
    ```text
    /nextphase
    ```
    *(Всегда реализуйте ровно одну фазу, проверяйте её, сохраняйте чекпоинт и останавливайтесь.)*
5.  **Финальная проверка и отправка**:
    ```text
    /shipcheck
    ```

## Главное правило

Не перескакивайте через фазы. Заявления LLM в чате не являются подтверждением выполнения задачи. Доверяйте только детерминированным командам, тестам, диффам и логам в локальном окружении.

Подробная документация находится в корневом файле [README.ru.md](../../README.ru.md) репозитория agentic-pipeline.
