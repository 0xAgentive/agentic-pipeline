# Using the ChatGPT Companion with This Project

The ChatGPT companion is the planning and audit layer. Antigravity is the execution layer in this workspace.

## Use the ChatGPT companion for

- writing specifications;
- reviewing other models' plans;
- preparing `/specdoc`, `/planonly`, `/nextphase` prompts;
- auditing logs, screenshots and artifacts;
- deciding whether `/shipcheck` is safe;
- planning pipeline migration.

## Use Antigravity for

- editing project files;
- running tests;
- executing workflows;
- producing evidence;
- publishing through git/gh when approved.

## Important rule

Do not migrate the pipeline in the middle of an active phase. Finish the current phase and run `/auditphase` first.
