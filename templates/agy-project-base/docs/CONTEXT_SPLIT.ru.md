# Как использовать ChatGPT companion вместе с этим проектом

ChatGPT companion — это слой планирования и аудита. Antigravity — слой исполнения в этом workspace.

## Используйте ChatGPT companion для

- формулирования ТЗ;
- анализа планов других моделей;
- подготовки `/specdoc`, `/planonly`, `/nextphase` prompt;
- аудита логов, скриншотов и артефактов;
- решения, можно ли запускать `/shipcheck`;
- подготовки миграции pipeline.

## Используйте Antigravity для

- изменения файлов проекта;
- запуска тестов;
- выполнения workflow;
- создания evidence;
- публикации через git/gh, если разрешено.

## Важное правило

Не мигрируйте pipeline посреди активной фазы. Сначала завершите текущую фазу и запустите `/auditphase`.
